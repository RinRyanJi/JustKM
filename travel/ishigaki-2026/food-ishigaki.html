🚨 資料來源限制 · 必讀

🔍 Maps POI 資料可信,但菜單真偽仍待親查
  • 🟢 店家存在 / 地址 / 評分 = Maps POI 已驗證(7/13 Playwright 抓)
  • 🟡 菜單 / 招牌菜 / 口味 = Maps 無資料,需打電話或到場看板確認 — 我們之前查到「具志堅くわっちぃー」上有人寫「鮪魚料理」、但 Maps 本身無菜單內容
  • 🔴 「幻之花 / Mariya Shake / 沖繩黑糖奶昔」這類詭異細節 = 不寫入,因為沒有 Maps 公開資料佐證
  • ⚠ Maps 未顯示「週五」(7/17 是週五)營業時間 — 此時段以「週四推估」算安全(週四營業 → 週五應也營業)· 但店家週二 ~週日為準,週三/週四都連續型較可靠

🔴 被排除的店家

下列 2 家「Maps 搜不到」,不寫進資料庫:
  • ❌ さがり花(或 さがりはな)— Maps POI 不存在 · 名稱疑「幻之花 / サイの花」被誤拼
  • ❌ とぅもーるショップ — Maps POI 不存在 · 名稱疑沖繩語「友⿰⿰⿰⿰⿰⿰· 賣店」的日文片假名拼寫錯誤

📋 為什麼排除:你 7/13 給的清單中 50%(2/4)Maps 找不到,與「只保留可驗證資料」原則不一致,故不收錄。可能這些店家用其他名稱營業(如「琉球◯◯」/「◯◯shop」),需親查或電訪。

📡 7/13 14:00 Maps 即時快照 (驗證時間)

本次 3 家「已打烊 · 週二」意味:
  • 今天是 週一 · Maps 抓的當下所有店家「已打烊」
  • 下一個營業日「週二」就是 明天 7/14 (二)
  • 今(7/13)Maps「週一」營業時間這欄不可信 — Maps 通常沒設週一營業(那麼多觀光區週一休息很常見)
  • ⚠ 週五 (7/17) 沒有 Maps 公開快照 — 因此表中的「09:30 / 06:30 / 08:00」是「Maps 通用營業時間」≠「7/17 當下營業時間」

📍 行程搭配 · 7/17 早午

🕗 08:00 抵石垣 → 09:00 入境 → 09:00-10:00 在離島碼頭內吃早午

你 7/17 必須搭 10:30 八重山觀光フェリー去大原(看 itinerary.html §2 確認),所以早餐 / 早午餐只能在 碼頭內 吃。

三家 營業時間 都包含 09:00-10:00:

  • ✅ 七人本舗 06:30 開始 → 輕食外帶(適合趕船 / 沒時間坐下來吃)
  • ✅ お食事処ソムリエ 離島ターミナル店 08:00 開始 → 定食坐下來吃(適合有時間慢慢吃)
  • ⚠ お食事処具志堅くわっちぃー 09:30 開始 → 你 09:00 才入境,這家 9:30 之後 才有開,時間很趕

推薦路徑:09:00-10:00 選「七人本舗」(軽食外帶) 或 「ソムリエ」(定食坐下吃) · 10:00 上船 · 不用額外跑,離島碼頭內全程完成。

📋 寫入/排除判定規則(資料庫原則):
  1. 🟢 Maps POI 上有「店家名 + 地址」且評分 ≥ 0 顆星 = 寫入
  2. 🟡 店家聲稱在「離島碼頭內」 需 POI 顯示地址 1 Misakicho 或標記 ishigaki terminal
  3. 🔴 沒有 Maps POI 但被 Gemini / 旅遊網站描述的店 = 不寫入,除非使用者親自去過
§ 01

三大招牌 · 來石垣不吃會後悔

MUST EAT

🥩 石垣牛燒肉

¥3,000-8,000/人 · 1.5 hr

石垣牛是和牛頂級品牌之一,僅次於神戶牛/松阪牛/近江牛。4 星/5 星餐廳人均 ¥4,000-8,000。
推薦:焼肉 島 (きたうち牧場直營) · 炭火焼肉 一二三 · まるみつ本店

🍜 八重山蕎麥麵

¥700-1,200 · 30 min

沖繩特有蕎麥麵,湯底用豚骨 + 柴魚,搭配紅薑/三枚肉。
推薦:岸本食堂 (創業 50+ 年) · 仲間 · 登野城 まるみつ · 八島蕎麥麵

🌶 邊銀食堂辣油

¥800-1,200 · 伴手禮首選

石垣島最具代表性伴手禮,邊銀食堂 (ペンギン食堂) 原創辣油。
機場免税 不賣 邊銀辣油 (太搶手),回市區採買;
機場可買:石垣牛 / 泡盛 / 蝦餅

⚠ 邊銀辣油購買時機

邊銀食堂本店位於市區,7/18-7/21 任何一天中午或下午都可買。
建議 7/21 (Day 6) 退房前最後採買,帶去機場 check-in 行李。
