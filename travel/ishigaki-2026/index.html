隨身 vs 託運
行動電源 + 貴重物品 = 隨身；超過 100ml 液體 = 託運

🎒 隨身 (登機)

  • 護照、錢包、手機 + 行動電源後者必須隨身
  • 重要文件影本 / 列印
  • 一套換洗衣物 + 盥洗包
  • 處方藥 + 過敏史卡片
  • 貴重物品、相機
  • 信用卡、現金少量

🧳 託運

  • 主要衣物、鞋子
  • 液體超過 100ml（防曬、化妝水）
  • 備用衣物
  • 紀念品空位（回程免稅先買）
  • 行動糧、零食、泡麵
  • 電子書 / 平板電源選一個

• 7/11 (六)：全航路終日欠航（颱風 9 號影響）

• 7/12 (日)：午前便欠航、午後便未定

• 動作：建議 7/14 與 7/16 當天 早上各掃一次上面的 ② 連結；若 7/17 全航路仍未恢復，改 backup plan。

🛑 ⚠ 船班復原 ≠ 島上巴士復原（西表島交通）

西表島巴士停駛與渡輪是 兩個獨立系統，不連動：

• 一般惡劣（強風/大雨）：船班有開 → 島上巴士照常營運（銜接港口旅客）

• 颱風 / 暴風警報發布：西表島交通巴士 全線運休（島上巴士營業所暫停）

• 警報解除後：必須等到縣道 215 號（島上唯一主幹道）無路樹倒塌/積水/土石流後才依序恢復

→ 陷阱：渡輪 7:30 有開 ≠ 8:00 巴士有開。解決方案：住宿訂前先問「是否有自家接送車」、使用飯店 tour 接送（浮潛/獨木舟都有巴士接送到港口集合）

📡 颱風季標準 SOP

1. 慢查船班 — 暴風前 12–24hr 船公司網站會公告。抵達石垣後到飯店請服務員幫忙查

2. 延誤 24hr+ — 旅平險「班機延誤險」啟動（需自付 ≥ 4hr 才賠）

3. 島間船停 — 西表/竹富活動取消時，退款約 7 個工作天，記得保留票根

4. 大災難 — 緊急聯絡：駐那霸辦事處 098-832-4056 / 外交部緊急 0800-085-095
